# Autoclicker

### Features:
 - Click interval in ms
 - Random click interval
 - Toggle clicks keyBind
 
 ### How to use?
 - Download **Autoclicker.exe**
 - Run **Autoclicker.exe**
 - Windows defender might pop up
 just click **More info** 
 and press **Run anyway** 
 - Write the keyBind _(eg. Home)_
